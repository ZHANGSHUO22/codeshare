欢迎来到我的个人知识库！这里记录了我学习编程过程中的笔记、心得和总结。

知识库采用类似GitBook的目录结构，方便查找和阅读。左侧是目录导航，点击可以快速跳转到相应章节。

这个网页完全使用HTML、CSS和JavaScript构建，没有使用任何框架，代码简洁易懂。

HTML基础

HTML（超文本标记语言）是构建网页的基础。

HTML文档结构

<!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <title>网页标题</title> </head> <body> <h1>这是一个标题</h1> <p>这是一个段落</p> </body> </html>

常用HTML标签

  • <h1>-<h6>：标题标签
  • <p>：段落标签
  • <a>：链接标签
  • <img>：图像标签
  • <ul>,<ol>,<li>：列表标签

CSS样式

CSS（层叠样式表）用于控制网页的样式和布局。

CSS选择器

  • 元素选择器：p { color: red; }
  • 类选择器：.classname { font-size: 16px; }
  • ID选择器：#idname { background: blue; }

盒模型

CSS盒模型包括内容(content)、内边距(padding)、边框(border)和外边距(margin)。

.box { width: 300px; padding: 20px; border: 1px solid #ccc; margin: 10px; }

JavaScript

JavaScript是一种脚本语言，用于为网页添加交互功能。

变量声明

// ES6之前的变量声明 var name = "张三"; // ES6引入的变量声明 let age = 25; const PI = 3.14;

函数定义

// 函数声明 function greet(name) { return "Hello, " + name; } // 箭头函数 const greet = (name) => { return "Hello, " + name; }
